## 2025


### A Hybrid Random Forest and CNN Framework for Tile-Wise Oil-Water Classification in Hyperspectral Images
**Authors:** S.Mohammad S.Ahmadi, Mehdi Nickzamir<br>
**Abstract:** This paper introduces a hybrid framework that improves oil-water classification in hyperspectral images by training a Convolutional Neural Network on Random Forest probability maps to integrate spatial context with pixel-wise spectral data.<br>
[PDF](/Download/2502.00232v1.pdf) | [arXiv](https://arxiv.org/html/2502.00232v1)

### Extending Chronos: Fine-Tuning for Financial Forecasting and Consistency Calibration
**Authors:** S.Mohammad S.Ahmadi, Mehdi Nickzamir, Erfan Bayat, Andrea Vasco Grieco<br>
**Abstract:** This work enhances the Chronos transformer model for financial forecasting by integrating domain-specific fine-tuning on stock and synthetic data with a perturbation-based consistency calibration (C3) method, resulting in a 15% increase in accuracy and more reliable probability estimates.<br>
[PDF](/Download/1752784092246.pdf) | [Code](https://github.com/smsag99/chronos-forecasting)

### Natural Language Queries in Egocentric Videos
**Authors:** S.Mohammad S.Ahmadi, Yasaman Noshirvanbaboli, Amirhossein Lotf Ranaei<br>
**Abstract:** This project enhances natural language query (NLQ) performance in egocentric vision by identifying the optimal combination of EgoVLP features and the VSLNet model, while further extending the framework with a fine-tuned Gemma 2B model for automated query generation.<br>
